# Radiology Center Booking System

A full-stack web system for managing radiology center appointments.

## My Role
- Full-Stack Developer (Team Member)

## Technologies Used
- **Backend:** C#, ASP.NET Core
- **Frontend:** Angular, TypeScript
- **Database:** SQL Server
- **Notifications:** Firebase

## Features I Worked On
- Appointment booking system
- Firebase Notifications for patient alerts
- User management and authentication
- Web control panel for center staff

## Project Context
This is a real-world system built for a radiology center. I worked as part of a team to develop and deliver the complete solution.
